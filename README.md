# Resume Screener Python 🐍

A **Python project** to automatically screen resumes and match them against a predefined set of skills. This project is designed to help recruiters quickly shortlist candidates based on key technical skills.

---

## **Key Features**

- ✅ Supports both **PDF** and **DOCX** resume formats  
- ⏱️ Automatically extracts text from resumes  
- 🏆 Matches candidate skills with **job description skills**  
- 📊 Calculates a **resume score** and shows **matched skills**  
- 🛠️ Exception handling ensures the program doesn’t crash on errors
